AI 기반 SEO 분석 & 콘텐츠 생성 웹앱

AI-powered SEO analysis & content generation web app

진행중 In Progress

SEO 키워드 분석의 복잡함을 AI로 해결하는 풀스택 웹앱입니다. 단순히 키워드만 분석하는 게 아니라, AI가 틈새 시장을 제안하고 SEO 최적화된 블로그 콘텐츠까지 자동으로 생성해주는 종합 솔루션입니다.

기술적 구현:
• React + TypeScript 프론트엔드 (Vite 번들러)
• Node.js + Express 백엔드 API 서버
• Google Gemini API로 AI 콘텐츠 생성 엔진 구축
• 네이버 검색광고 API 연동으로 실시간 키워드 데이터 수집
• CORS 정책 및 API 키 보안 처리를 위한 분리형 아키텍처
핵심 알고리즘:
• AI 키워드 분석: 검색량, 경쟁강도, 수익성을 종합한 점수 산출
• 콘텐츠 자동 생성: 키워드별 맞춤 SEO 최적화 글 작성
• 실시간 트렌드 분석: 네이버 API 데이터를 AI가 해석해 인사이트 제공
• 다양한 분석 모드: 직접 분석, API 연동, 혼합 분석 지원

A full-stack web app that solves the complexity of SEO keyword analysis with AI. Rather than just analyzing keywords, it's a comprehensive solution where AI suggests niche markets and automatically generates SEO-optimized blog content.

Technical Implementation:
• React + TypeScript frontend (Vite bundler)
• Node.js + Express backend API server
• AI content generation engine built with Google Gemini API
• Real-time keyword data collection via Naver Search Ads API integration
• Decoupled architecture for CORS policy and API key security handling
Core Algorithms:
• AI keyword analysis: Comprehensive scoring combining search volume, competition, and profitability
• Automated content generation: Customized SEO-optimized writing for each keyword
• Real-time trend analysis: AI interprets Naver API data to provide insights
• Multiple analysis modes: Direct analysis, API integration, hybrid analysis support
