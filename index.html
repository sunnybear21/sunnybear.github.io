About Me

안녕하세요! 저는 다양한 AI들을 오케스트레이션해서 실제 서비스를 만드는 AI 빌더입니다.

GPT만 쓰는 게 아니라 Gemini, Claude, Suno, Veo 등 각각의 강점을 믹스해서 활용하는 걸 좋아해요! 한 AI에 의존하지 않고 상황에 맞는 최적의 AI를 선택해서 협업하는 스타일입니다.

비전공자지만 3주 만에 모바일 앱 출시, TTS 음성 기능 구현, 4종 미니게임 개발 등 실제로 작동하는 프로덕트를 계속 만들어내고 있습니다!

날씨요정 달멍씨 v1.9

TTS 음성 + 미니게임이 있는 감성 날씨 앱

출시완료

코딩 지식 제로에서 시작해서 3주 만에 Google Play Store 정식 출시한 모바일 앱입니다. 단순한 날씨 앱이 아니라 TTS 음성으로 달멍이가 직접 말해주고, 4가지 미니게임까지 즐길 수 있는 종합 엔터테인먼트 앱으로 진화시켰습니다.

주요 도전과제:
• MIT App Inventor → Kodular → Android Studio로 3번의 플랫폼 전환
• GitHub API 키 노출 보안 사고 및 해결
• SDK35 정책 대응한 권한 시스템 구현
• TTS 엔진 통합 및 캐릭터별 음성 패턴 개발
• 4종 미니게임 로직 설계 및 메모리 최적화
해결 과정:
• Firebase + Render 서버로 API 키 보안 아키텍처 구축
• 구글 플레이 정책에 맞춘 권한 요청 플로우 설계
• 다양한 AI(GPT, Gemini)와 협업해 복잡한 게임 로직 구현
• 20명 비공개 테스트 통과 후 정식 출시 성공
Android Studio Java TTS API Weather API Firebase Render
앱 다운로드

달멍봇 키워드 마스터 V2

AI 기반 SEO 분석 & 콘텐츠 생성 웹앱

진행중

SEO 키워드 분석의 복잡함을 AI로 해결하는 풀스택 웹앱입니다. 단순히 키워드만 분석하는 게 아니라, AI가 틈새 시장을 제안하고 SEO 최적화된 블로그 콘텐츠까지 자동으로 생성해주는 종합 솔루션입니다.

기술적 구현:
• React + TypeScript 프론트엔드 (Vite 번들러)
• Node.js + Express 백엔드 API 서버
• Google Gemini API로 AI 콘텐츠 생성 엔진 구축
• 네이버 검색광고 API 연동으로 실시간 키워드 데이터 수집
• CORS 정책 및 API 키 보안 처리를 위한 분리형 아키텍처
핵심 알고리즘:
• AI 키워드 분석: 검색량, 경쟁강도, 수익성을 종합한 점수 산출
• 콘텐츠 자동 생성: 키워드별 맞춤 SEO 최적화 글 작성
• 실시간 트렌드 분석: 네이버 API 데이터를 AI가 해석해 인사이트 제공
• 다양한 분석 모드: 직접 분석, API 연동, 혼합 분석 지원
